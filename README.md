# bridge-business-cost-nav

株式会社BRIDGEの「事業者向けコスト削減・資金繰り確認ナビ」です。

## 目的

このページは、事業者・個人事業主・法人が、固定費、資金繰り、会計、通信、決済、業務効率化などの外部サービスを確認する前に、悩み別に選択肢を整理するための情報ページです。

## 方針

- アフィリエイト広告を掲載できる構成
- ただし、BRIDGEが直接サービス提供しているような表現は避ける
- 申込、審査、契約、料金、税務・会計判断はリンク先または専門家で確認する
- 「必ず」「絶対」「審査に通る」「安くなる保証」などの断定表現は使わない
- スマホで見やすい選択式ナビにする
